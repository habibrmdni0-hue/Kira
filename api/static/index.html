🖥️ AMD MI300X — Inferensi 70B

Kira menggunakan model bahasa 70 miliar parameter (Llama-3.1-70B / Qwen2.5-72B) untuk analisis bisnis mendalam. Model ini dijalankan di GPU AMD MI300X melalui endpoint vLLM yang kompatibel dengan OpenAI API — tanpa perubahan kode apa pun.

Reasoning Agent memanggil model 70B ini setiap ada alert bisnis yang butuh analisis: mengapa situasinya terjadi, dampak finansialnya, dan satu langkah konkret untuk pemilik warung.

Endpoint dikonfigurasi via REASONING_BASE_URL di environment. Ganti URL ke AMD Developer Cloud atau Fireworks AI — tidak ada perubahan kode yang diperlukan.
